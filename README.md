# 🛹 Skate Dice

> Aplicação desktop desenvolvida em Python para gerar manobras aleatórias de skate, simulando a dinâmica do "Skate Dice" e auxiliando no treino de forma criativa e desafiadora.

---

## 🚀 Preview

- 📌 Geração aleatória de manobras solo
- 📌 Modos de treino em caixote (níveis progressivos)
- 📌 Combinação de base, direção, entrada e saída
- 📌 Sistema de “chances” para simular tentativas reais

(Preview)[./preview.gif]

---

## 💡 Contexto do projeto

O projeto foi criado com base em uma prática comum dentro do skate chamada **"Skate Dice"**, onde manobras são sorteadas aleatoriamente para treino.

Como praticante, percebi que os aplicativos existentes eram limitados, com poucas variações de manobras e baixa profundidade nas combinações.

A partir disso, desenvolvi minha própria versão da ferramenta, com maior variedade e possibilidades, tornando o treino mais dinâmico e desafiador.

Este projeto também marcou uma fase importante do meu aprendizado, sendo uma das primeiras aplicações onde consegui estruturar lógica de randomização mais complexa.

Além disso, ele foi pensado como base para evolução futura, com a intenção de ser transformado em uma aplicação mobile mais completa.

## 📚 Conceitos aplicados

- Geração de valores aleatórios (`random`)
- Estruturação de lógica condicional
- Criação de interfaces com CustomTkinter
- Organização de dados com listas
- Combinação dinâmica de elementos (base + direção + manobra + variações)
- Simulação de mecânicas reais de treino

---

## ⚙️ Funcionalidades

- ✅ Geração de manobras solo aleatórias
- ✅ Modos de treino em caixote com diferentes níveis de dificuldade
- ✅ Combinação de múltiplos fatores (base, direção, entrada e saída)
- ✅ Sistema de chances (tentativas)

## ▶️ Como executar o projeto

* Python 3 instalado

### Instalação da dependência


```bash

git clone https://github.com/vitordrs/Skate-dice.git

# Acesse a pasta
cd Skate-dice

python nome_do_arquivo.py
```
